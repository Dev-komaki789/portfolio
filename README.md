# portfolio

エンジニアとしての自己紹介・スキル・制作物をまとめたシングルページのポートフォリオサイト。

## 含まれている情報

- プロフィール / 自己紹介（経歴と現在の学習状況）
- スキル（実務 / 構築 / 学習中の 3 段階で分類）
- 制作物（WMS / EC サイト / その他の小品）
- 連絡先（GitHub / メール）

## 技術スタック

React 19 + TypeScript + Vite + Tailwind CSS 4

## メモ

コンテンツは `src/content.ts` に集約しており、文章・スキル一覧・制作物リスト・
連絡先などをこのファイル 1 つで編集できる構造にしている。
