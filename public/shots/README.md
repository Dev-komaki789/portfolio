# スクリーンショットの置き場所

このフォルダに画像ファイルを置くだけで、作品モーダルのギャラリーに表示されます。
コードを編集する必要はありません（`vercel` も自動で反映します）。

## ファイル名の対応（`src/content.ts` の `images` で指定済み）

| 作品 | 置くファイル名 |
| --- | --- |
| 倉庫管理システム (WMS) | `wms-1.png` 〜 `wms-4.png` |
| EC サイト | `ec-1.png` 〜 `ec-3.png` |
| サイバーパンク風メモ帳 | `notepad-1.png` `notepad-2.png` |
| 4択クイズアプリ | `quiz-1.png` `quiz-2.png` |

- 画像がまだ無いファイルは、自動で `placeholder.svg`（準備中の絵）が表示されます。
- 枚数を増やす／減らすときは `src/content.ts` の各作品の `images: [...]` を編集してください。
- 形式は `.png` / `.jpg` どちらでもOK（`content.ts` の拡張子と合わせる）。
- 横長（例: 1600×1000 程度）だと見栄えが揃います。
